### Package description
This package helps improve your brain with some games. Game today has 5 games: brain-calc, brain-even, brain-gcd, brain-prime, brain-progression.

brain-games - welcome user. This function has in all games.
brain-calc - sum or multiply random numbers.
brain-even - define even number.
brain-gcd - find gcd 2 numbers.
brain-prime - define prime number.
brain-progression - find missed number in progression row.

### Setup

```bash
make install
make build
make package-install
```

### Example

You can start a game by typing one of the following commands: brain-calc, brain-even, brain-gcd, brain-prime, or brain-progression. For example, to run the welcome function, type: brain-games.

```bash
brain-games

Welcome to the Brain Games!
May I have your name? Alex

Hello, Alex!
```
